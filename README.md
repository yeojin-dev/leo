## How To Use

### Requirements

* python 3.8
* mongodb 4.2
* .env.template 파일을 활용해 .env 파일을 생성

### Run in local env.

파이썬 가상환경에서 아래 스크립트 사용

```bash
$ python app.py
```

### Run a Docker container

도커 이미지 빌드는 아래 스크립트 사용

```bash
$ docker build --tag leo:latest --rm .
```

빌드 이후 컨테이너 실행

```bash
$ docker run -d -p 5000:5000 leo:latest
```
